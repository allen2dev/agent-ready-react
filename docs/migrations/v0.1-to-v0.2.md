# Migration: v0.1 alpha → v0.2 beta

## Breaking changes

**None** for runtime JSON APIs.

## React API clarifications (documentation alignment)

### `useAgentAction` — explicit `handle` (required)

v0.2 documents the **implemented** signature (also used in playground and tests):

```tsx
const handle = "app://forms/contact/main" as const;

useAgentSurface({ handle, title: "Form", capabilities: ["act"] });

useAgentAction(handle, submitForm, async (input) => {
  return { ok: true };
});
```

If your code assumed implicit handle binding from context only, add the `handle` argument as the first parameter.

### `useAgentObservation` — explicit `handle` (required)

```tsx
useAgentObservation(handle, observationDef, () => ({ count }), { debounceMs: 100 });
```

## New capabilities in v0.2

- `runtime.readObservation` / `registerObservation`
- `useAgentObservation`, `useAgentCatalog`
- `@agent-ready/bridge` HTTP JSON-RPC
- `@agent-ready/observability`, `@agent-ready/devtools` (optional)

## Upgrade steps

```bash
pnpm add @agent-ready/react@beta @agent-ready/runtime@beta @agent-ready/schema@beta
```

1. Pass `handle` as the first argument to `useAgentAction` and `useAgentObservation`.
2. Optionally add `@agent-ready/bridge` for HTTP agent access.
3. Run `pnpm test` in your app after upgrading.
