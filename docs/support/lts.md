# Long-Term Support (LTS)

## Support window

- **Patch releases** (bug/security fixes) for the **current GA minor** for **6 months** after the next minor GA ships.
- Example: when `1.1.0` releases, `1.0.x` receives patches for six months.

## Supported lines

| Line | npm tag | Support |
|------|---------|---------|
| 1.0.x | `latest` | Active GA |
| 0.3.x | `rc` | Until 1.0 GA + 6 months patches |
| 0.2.x | `beta` | Best-effort security only |
| 0.1.x | — | EOL |

## Reporting issues

Open GitHub issues with reproduction, SDK version, and policy/audit configuration (redact secrets).
